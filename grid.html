css grid & flexbox layout practice

Two layout exercises for a simple page shell — header, profile column, intro, logo, main content, and footer. The colored blocks make it easy to see how each region stretches and aligns.

Note: I did not write the CSS for the flexbox example, but rather generated it using Cursor. My hope is to learn how to reverse engineer how it is working by analyzing the separate stylesheet.

What each row does

The outer grid-demo-container is a column flexbox. The header sits on top. The introduction and main rows are nested flex rows — profile and logo use flex: 0.25 for a narrow side column; intro and content use flex: 1 to fill the rest.

The footer is a separate row below the container, centered with justify-content: center and spaced social placeholders.
