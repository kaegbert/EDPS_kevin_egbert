How classes work on this site

A class is a label on an HTML element. CSS targets that label to apply styles. JavaScript can also use classes — for example .reveal and .nav-toggle in script.js.

All portfolio pages share one stylesheet: styles.css. learning-html/ is separate and does not use it.

Multiple classes on one element

<header class="site-header sidebar">
<a class="nav-logo logo-mark">
<a class="project-feature reveal">
<a class="nav-link is-active">

Each class adds rules; they stack together on the same element.

Classes vs IDs

Class (.name)
Reusable on many elements. Example: many nav-link items.
ID (#name)
One per page. Example: id="main" for the skip link target.
