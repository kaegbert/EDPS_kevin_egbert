project cards — inline to classes

I started with inline style="..." on every row and card. I moved the repeated rules into styles.css and kept the HTML shorter with classes.

.card-stack — I use this to stack the three card rows vertically. It replaces display: flex; flex-direction: column; gap: 2.5rem; margin-bottom: 2.5rem;

.justify-left, .justify-center, .justify-right — I pair these with .flex-row to move each card left, center, or right. They replace inline justify-content on each row.

.card-placeholder — I use this with .card for grey placeholder projects. It replaces inline color, margin-bottom, and text-decoration: none on the title. 

<!-- before -->
<div style="display: flex; justify-content: center;">
  <div class="card">
    <strong style="text-decoration: none;">project two</strong>
  </div>
</div>

<!-- after -->
<div class="flex-row justify-center">
  <div class="card card-placeholder">
    <strong>project two</strong>
  </div>
</div>

column layout — inline to classes

I built the column demo the same way — three classes instead of repeating inline flex and box styles.

.flex-row — I use this for the outer row of three columns. It replaces display: flex; flex-direction: row; gap: 16px;

.flex-column — I use this for each vertical stack inside the row. It replaces display: flex; flex-direction: column; flex: 1; gap: 16px;

.column-box — I use this for each numbered box. It replaces border: 1px solid #222222; padding: 12px; min-height: 80px; 

<!-- before -->
<div style="display: flex; flex-direction: row; gap: 16px;">
  <div style="display: flex; flex-direction: column; flex: 1; gap: 16px;">
    <div style="border: 1px solid #222222; padding: 12px; min-height: 80px;">1</div>
  </div>
</div>

<!-- after -->
<div class="flex-row">
  <div class="flex-column">
    <div class="column-box">1</div>
  </div>
</div>

Now I can edit a class once in styles.css and every element with that class updates. I can also combine classes on one element — for example class="flex-row justify-center" or class="card card-placeholder".
