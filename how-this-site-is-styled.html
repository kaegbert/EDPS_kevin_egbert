Live setup — classes in styles.css

Every portfolio page links to styles.css. Flexbox only — no CSS grid, no JavaScript, no mobile hamburger. The shell uses four classes on every page; the flexbox learning page adds more. About still uses inline flex for the portrait layout.

Shell (every page)

Class Purpose
layout Flex row — sidebar + main (display: flex, min-height: 100vh)
sidebar Fixed 260px left column (flex-shrink: 0)
main flex: 1 content area (padding: 60px 5%)
card Project cards (max-width 42rem); child selectors style strong, p, img, .placeholder

Flexbox helpers (learning-to-build-use-flexbox.html)

Class Purpose
flex-row Horizontal flex row, gap: 16px
flex-column Vertical stack inside a row; flex: 1
column-box Bordered box in the column demo
card-stack Vertical stack of card rows
justify-left / justify-center / justify-right Pair with .flex-row to align a card
card-placeholder Modifier with .card for placeholder projects (no link, no underline on title)

Combine classes on one element, e.g. class="flex-row justify-center" or class="card card-placeholder".

Inside a project card

Cards use one class on the outer element. Inner parts use plain HTML tags styled in CSS: 

<a class="card" href="...">
  <strong>Title</strong>     <!-- .card strong in styles.css -->
  <p>Subtitle</p>         <!-- .card p -->
  <img src="...">         <!-- .card img -->
  <span class="placeholder"> <!-- .card .placeholder -->
</a>

Modifier pattern: class="card" on the outer element plus class="placeholder" on the span when there is no screenshot. Placeholder styling comes from .card .placeholder in styles.css.

Inline styles

About portrait layout and hint text on this page still use inline style="...". The flexbox learning page mixes classes from styles.css with inline notes. 

<div class="flex-row justify-center">
<p style="color: #999999; margin: 0 0 24px;">

Classes and inline styles can stack on the same tag, e.g. class="card" style="margin-bottom: 0;".

Classes vs IDs

Class (.name)
Reusable on many elements. Example: every page has class="layout".
ID (#name)
One per page. This simplified site uses few IDs; most layout is classes + inline styles.

How each page works

Page and section titles — Use plain <h1> for the page title and <h2> for section headings inside main. Edit both in styles.css: 
.main h1 { font-size: 24px; font-weight: 700; ... }
.main h2 { font-size: 20px; font-weight: 700; ... }
On other pages, <h1> may be commented out; uncomment to show a page title. The flexbox page uses <h2> for “column layout”.
Home (index.html) — Neolithia project card, then an h2 “blog / learning documentation” section with two more card links. Headings use .main h1 / .main h2 from styles.css. 
<h1>projects</h1>
<a class="card" href="neolithia/">...</a>
<h2>blog / learning documentation</h2>
<a class="card" href="learning-to-build-use-flexbox.html">...</a>
About (about.html) — Same layout / sidebar / main shell as home. The portrait sits centered above the text using inline flexbox (not a CSS class): 
<div style="display: flex; flex-direction: column; align-items: center; gap: 32px;">
  <figure style="width: 100%; max-width: 42rem;">...</figure>
  <div style="width: 100%;">...</div>
</div>
Change max-width on the figure to resize the portrait.
Learning to build using flexbox (learning-to-build-use-flexbox.html) — Project cards use card-stack, flex-row, justify-left / justify-center / justify-right, and card card-placeholder. The column demo uses flex-row, flex-column, and column-box. First-person notes under each demo explain the inline-to-class changes. 
<div class="flex-row justify-center">
  <div class="card card-placeholder">...</div>
</div>

<div class="flex-row">
  <div class="flex-column">
    <div class="column-box">1</div>
  </div>
</div>
