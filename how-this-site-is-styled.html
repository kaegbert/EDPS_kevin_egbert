What is a class? — A class is a label on an HTML element. CSS targets that label to apply styles. Example: .sidebar in styles.css styles every element with class="sidebar".

Live setup (5 CSS classes only)

All portfolio pages share styles.css. learning-html/ is separate and does not use it.

Class Purpose
layout Flex row — sidebar + main (display: flex)
sidebar Fixed 260px left column (flex-shrink: 0)
main flex: 1 content area

Inside a project card

Cards use one class on the outer element. Inner parts use plain HTML tags styled in CSS: 

<a class="card" href="...">
  <strong>Title</strong>     <!-- .card strong in styles.css -->
  <p>Subtitle</p>         <!-- .card p -->
  <img src="...">         <!-- .card img -->
  <span class="placeholder"> <!-- .card .placeholder -->
</a>

Modifier pattern: class="card" on the link plus class="placeholder" on the span when there is no screenshot.

Multiple classes on one element 

<span class="placeholder" role="img">

Each class adds rules; they stack on the same element. You can also combine a class with inline style="..." on the same tag.

Classes vs IDs

Class (.name)
Reusable on many elements. Example: every page has class="layout".
ID (#name)
One per page. This simplified site uses few IDs; most layout is classes + inline styles.

How each page works

Home (index.html) — The shell uses 5 classes total in styles.css: layout, sidebar, main, note, card. 
<div class="layout">     <!-- display: flex; flex-direction: row -->
  <aside class="sidebar">  <!-- width: 260px; flex-shrink: 0 -->
  <main class="main">      <!-- flex: 1 (fills remaining width) -->
Try adding style="gap: 24px;" on .layout in index.html to see flex gap.
About (about.html) — Same layout / sidebar / main shell as home. The portrait sits centered above the text using inline flexbox (not a CSS class): 
<div style="display: flex; flex-direction: column; align-items: center; gap: 32px;">
  <figure style="width: 100%; max-width: 42rem;">...</figure>
  <div style="width: 100%;">...</div>
</div>
Change max-width on the figure to resize the portrait.
Learning to build use flexbox (learning-to-build-use-flexbox.html) — All three cards share one flex row so they sit side by side. Edit the container’s justify-content, gap, or each card’s flex inline: 
<div style="display: flex; flex-direction: row; gap: 16px; flex-wrap: wrap;">
  <a class="card" style="flex: 1 1 0; min-width: 12rem; max-width: none;">...</a>
  <div class="card" style="flex: 1 1 0; min-width: 12rem; max-width: none;">...</div>
</div>
max-width: none overrides the global .card cap so three cards can share one row.
Column layout (same page) — Three columns, three boxes each. All flexbox, inline on that page: 
<div style="display: flex; gap: 16px;">           <!-- row of columns -->
  <div style="display: flex; flex-direction: column; flex: 1; gap: 16px;">
    <div style="border: 1px solid #222; padding: 12px;">1</div>
  </div>
</div>
Try flex-direction: column on the outer div to stack columns vertically.
